# Tech Stack & Libraries

## Frontend Core
- **Vite**: Ultra-fast build tool and development server.
- **React 18**: UI rendering.
- **Tailwind CSS v3**: Utility-first CSS framework for ultra-fast styling.
- **Zustand**: Lightweight global state management.

## Visualization & Editor
- **Dagre**: Directed graph layout algorithm used to calculate X/Y coordinates for tables and relationship routing.
- **@monaco-editor/react**: The Microsoft Monaco editor (powers VS Code) brought to the browser.
- **Lucide-React**: SVG icon library. Icons are dynamically loaded via the `lucide-react` dynamic exports based on user strings.

## Export Libraries
- **html-to-image**: Renders DOM nodes to Canvas/PNG/SVG.
- **gifshot**: Encodes an array of base64 image frames into an animated GIF.
- **lz-string**: Super fast LZ compression used to serialize the DSL code into the URL hash for shareable links.
